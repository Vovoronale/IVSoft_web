IV GeoJSON

DXF → GeoJSON із прив’язкою до карти для GIS/державних сервісів.

Для чого

У кресленнях DXF часто немає точних координат, а підготовка правильного GeoJSON “вручну” займає багато часу. Це особливо відчутно, коли потрібно сформувати дані для GIS та державних сервісів (зокрема для форматів GeoJSON для ЄДСББ).

Як це працює (кроки 1–4)

  1. Завантажте DXF із контуром/ділянкою/об’єктом.
  2. Оберіть замкнені контури (полілінії), які потрібно конвертувати.
  3. Задайте опорні точки на кресленні та на карті для прив’язки.
  4. Отримайте GeoJSON з коректними координатами для подачі в GIS/державні сервіси.
